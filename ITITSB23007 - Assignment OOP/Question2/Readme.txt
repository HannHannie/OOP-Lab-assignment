#649 Lottery Project

#Project
 This project simulates a lottery system based on the popular "649 Lottery"

##How to use it
1. Compile the Java files: javac *.java
2. Run the program in Terminal.

##Interesting new feature of it:
1. Calculate the cost of the number of tickets you play.
2. Calculate total money won.
3. Calculate the profit/loss automatically.
